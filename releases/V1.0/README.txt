# YouTube Downloader by EugeneCyber

Данное приложение выполняет функцию загрузки видео с сервиса YouTube в качестве 1080 (если видео имеет качество хуже указанного, то будет загруженно максимально доступное).

### Пошаговая установка

0. Первые два шага необходимо сделать только при первом использовании приложения.
1. Запустить файл install_codec.bat от имени администратора и дождаться завершения установки. В случае, если установщик выдает ошибку "Невозможно загрузить файл ...\ffmpeg.ps1, так как выполнение сценариев отключено в этой системе", необходимо открыть PowerShell от имени администратора (Пуск -> PowerShell -> Запустить от имени администратора) и ввести команду "Set-ExecutionPolicy RemoteSigned" без кавычек. В открывшемся меню нажимаем английскую "A", применяя изменения ко всем пользователям.
2. Перезагрузить компьютер.
3. Запустить файл YouTube Downloader.exe.
4. В появившееся окно вставить ссылку на youtube-видео.
5. Видео загрузится в папку download в той же директории, что и исполняемый файл.
